### Week 2: AI 영상 생성 (4-5일)

#### Scene 1 생성 (고독 씬) - 1일
**목표:** 8초 분량, 감정적 훅

**생성 전략:**
1. Sora 2로 3가지 버전 생성:
   - Ver A: 더 어두운 톤 (극적)
   - Ver B: 부드러운 톤 (서정적)
   - Ver C: 다큐멘터리 스타일 (사실적)

2. 각 버전 평가 기준:
   - 감정적 임팩트 (가장 중요)
   - 캐릭터 얼굴 일관성
   - 조명의 자연스러움
   - 카메라 무브먼트 부드러움

3. 최종 선택 또는 베스트 파트 합성

**예상 시행착오:**
- 3-5회 재생성 필요 (얼굴 일관성)
- 조명이 너무 어둡거나 밝을 수 있음
- 카메라 무브먼트 속도 조정 필요

**Tip:**
- 같은 프롬프트로 seed 값만 바꿔 여러 번 생성
- 가장 좋은 표정의 프레임 캡처해서 참조로 활용

#### Scene 2A 생성 (태블릿 UI) - 1일
**목표:** 4초 분량, 기술 신뢰도

**생성 전략:**
1. Veo 3 사용 (UI 정확도 높음)
2. UI 디자인 먼저 Figma에서 제작 (선택사항)
3. 프롬프트에 UI 요소 정확히 명시

**UI 디자인 가이드:**
```
앱 이름: 잇다 (ITDA)
메인 컬러: #FF8C42 (따뜻한 오렌지)
서브 컬러: #7FB685 (부드러운 그린)
폰트: Noto Sans KR (한글), Inter (영문)
아이콘: 라운드 스타일, 친근함 강조
```

**대안:**
- UI 부분만 After Effects로 별도 제작 후 합성
- 실제 태블릿 촬영 후 AI 배경과 합성

#### Scene 2B 생성 (영상통화) - 1일
**목표:** 8초 분량, 인간적 연결

**생성 전략:**
1. 두 인물 따로 생성 후 편집에서 split-screen 합성
   - 노인 파트: Sora 2 (감정 표현 우수)
   - 학생 파트: Sora 2 또는 Veo 3

2. 또는 split-screen을 프롬프트에 명시하여 한 번에 생성

**연기 디렉션 (프롬프트에 반영):**
- 노인: 처음 조심스럽다가 점점 편안해짐
- 학생: 진심 어린 경청, 과하지 않은 리액션
- 타이밍: 대화 리듬 자연스럽게 (실제 대화처럼)

**AI 오버레이 그래픽:**
- After Effects로 별도 제작 추천
- 심플하고 깔끔하게 (건강 아이콘, 맥박 등)
- 투명도 30-40%로 방해되지 않게

#### Scene 3 생성 (몽타주 + 엔딩) - 2일
**목표:** 9초 분량, 임팩트와 여운

**Day 1: 몽타주 씬 (6초)**
1. 4-6개 개별 씬 생성:
   - 운동하는 노인
   - 요리하는 노인
   - 사진 보여주는 노인
   - 웃는 노인들 클로즈업

2. 각 씬 1.5초 내외로 생성 (여유있게 3초 생성 후 트림)

**Day 2: 로고 & 엔딩 (3초)**
1. 배경: 마지막 웃는 얼굴에서 디포커스
2. 로고는 별도 제작:
   - Adobe Illustrator 또는 Figma
   - 애니메이션은 After Effects
3. 텍스트 자막 명확하게

**로고 디자인 상세:**
```
잇다 (ITDA)
- 한글: 부드러운 라운드 폰트
- 심볼: 두 손이 닿을 듯 말 듯 + 하트 또는 불빛
- 색상: 그라디언트 (#FF8C42 → #FFB6A3)
- 크기: 화면의 25% 정도
- 애니메이션: 1초 페이드인 + 글로우 효과
```

### Week 3: 후반작업 (3-4일)

#### Day 1-2: 편집 (Premiere Pro / Final Cut Pro)

**타임라인 구성:**
```
00:00 - 00:08 | Scene 1 (고독)
00:08 - 00:12 | Scene 2A (태블릿)
00:12 - 00:20 | Scene 2B (영상통화)
00:20 - 00:26 | Scene 3A (몽타주)
00:26 - 00:30 | Scene 3B (로고/엔딩)
```

**전환 효과:**
- Scene 1 → 2A: 크로스 디졸브 (0.5초)
- Scene 2A → 2B: 컷 또는 빠른 디졸브 (0.3초)
- Scene 2B → 3A: 밝기 증가하며 컷
- 몽타주 내부: 빠른 컷 또는 whip pan
- Scene 3A → 3B: 디포커스 블러 전환

**편집 팁:**
- J-cut, L-cut 활용 (오디오가 영상보다 먼저/늦게)
- 음악 비트에 맞춰 컷 타이밍
- 30.00초 정확히 (프레임 단위로 체크)

#### Day 2-3: 컬러 그레이딩 (DaVinci Resolve)

**씬별 색감 전략:**

**Scene 1 (고독):**
```
LUT: Kodak 2383 Cool 또는 직접 조정
- Lift: Blue +10 (그림자에 차가움)
- Gamma: Saturation -15 (채도 낮춤)
- Gain: Slight warm (+5) to contrast
- Overall: 차갑고 쓸쓸한 느낌
```

**Scene 2 (연결):**
```
LUT: Warm transitional
- Lift: Neutral to slight warm
- Gamma: Saturation 0 to +10 (점진적 증가)
- Gain: Warm orange/pink in highlights
- Overall: 따뜻함으로 전환
```

**Scene 3 (희망):**
```
LUT: Golden hour / Kodak Portra 400
- Lift: Warm shadows
- Gamma: Saturation +20
- Gain: Golden highlights (#FFD700 톤)
- Overall: 밝고 희망적
```

**통일성 유지:**
- 모든 씬에 subtle film grain 추가 (일관성)
- 블랙 레벨 통일 (RGB 16, 16, 16)
- 화이트 포인트 통일 (RGB 235, 235, 235)

#### Day 3-4: 사운드 디자인

**음악 선곡:**

**추천 사이트:**
- Epidemic Sound (구독형, 고품질)
- Artlist (연간 구독)
- Musicbed (프리미엄)
- Uppbeat (무료 옵션 있음)

**음악 구조:**
```
00:00-00:08 | 미니멀 피아노, 쓸쓸한 분위기
            | 악기: 피아노 + subtle pad
            | BPM: 60-70, 느린 템포

00:08-00:20 | 따뜻한 전환, 어쿠스틱 기타 추가
            | 악기: 피아노 + 기타 + 스트링
            | 점진적 빌드업

00:20-00:30 | 클라이맥스, 희망찬 오케스트라
            | 악기: 풀 오케스트라
            | BPM: 80-90, 에너지 상승
            | 30초에서 resolution (해결)
```

**사운드 이펙트:**
- 00:09 - 태블릿 알림음 (부드러운 chime)
- 00:12 - 영상통화 연결 톤
- 00:21 - 몽타주 시작 whoosh
- 00:27 - 로고 등장 swoosh + sparkle
- 배경: subtle ambient (시계 째깍, 새소리 등)

**믹싱 레벨:**
```
음악: -12dB to -18dB (자막 읽을 수 있게)
효과음: -20dB to -24dB (자연스럽게)
환경음: -30dB to -35dB (분위기용)

다이나믹스:
- 음악은 감정 곡선 따라 페이드 인/아웃
- 중요한 순간(얼굴, 메시지)에서 음악 살짝 낮춤
- 30초 끝에서 깔끔한 페이드아웃
```

**자막 타이밍:**
```

00:27-00:30 | "혼자가 아닙니다 / You Are Not Alone"
              + 통계 (85% 감소 | 100% 대응)
```

**자막 디자인:**
```

크기: 최소 36pt (가독성)
색상: White (#FFFFFF)
외곽선: Black 3px stroke (가독성)
또는: 반투명 검은 박스 배경 (80% opacity)
애니메이션: 페이드인 0.3s, 홀드, 페이드아웃 0.3s
위치: 하단 safe area (화면 하단 20%)
```

## 🎨 추가 제작 팁

### After Effects 활용 (선택사항)

**로고 애니메이션:**
1. Shape layer로 손 두 개 그리기
2. 손가락 사이에 파티클 이펙트 (CC Particle World)
3. 텍스트 애니메이션: Fade + Scale (100% → 110% → 100%)
4. 전체에 Glow 효과

**UI 오버레이:**
1. 건강 아이콘 애니메이션 (Shape layer)
2. 맥박 라인 (Stroke 애니메이션)
3. 숫자 카운터 (Numbers 이펙트)
4. 투명도 조절로 자연스럽게

### 색보정 심화 (DaVinci Resolve)

**전문가 워크플로우:**
1. **1차: 색온도 보정** (Primaries)
   - 전체 톤 맞추기
   - 화이트 밸런스 조정

2. **2차: Look 적용** (LUT)
   - 영화 느낌 LUT 적용
   - 강도 50-70% (자연스럽게)

3. **3차: 세부 조정** (Curves)
   - RGB 커브로 콘트라스트
   - 그림자/하이라이트 디테일

4. **4차: 포인트 보정** (Qualifier)
   - 얼굴 톤 보정
   - 특정 색상 강조

5. **5차: Vignette & Grain**
   - 미묘한 비네팅
   - Film grain (2-5% 강도)

### 음악 편집 고급 기법

**음악 구간 매칭:**
1. 음악의 intro, verse, chorus, outro 파악
2. 영상의 감정 곡선과 매칭
3. 필요시 음악 편집 (구간 삭제/반복)
4. 30초에 맞춰 outro가 자연스럽게 끝나도록

**비트 싱크:**
- 중요한 컷을 음악 비트에 맞추기
- 몽타주 씬은 특히 리듬감 중요
- 로고 등장을 음악 클라이맥스에

## 🏆 대상 수상 전략 정리

### 차별화 포인트 강조

1. **기술 + 감성 균형**
   - AI 기술 설명 (신뢰도)
   - 인간적 스토리 (감동)
   - 둘의 조화가 핵심

2. **세대 연결 콘셉트**
   - 단순 케어가 아닌 "잇기"
   - 노인과 청년 모두 혜택
   - 지속가능한 사회적 가치

3. **브랜드 정체성**
   - 명확한 네이밍 (잇다)
   - 슬로건 ("혼자가 아닙니다")
   - 일관된 비주얼

4. **데이터 기반 신뢰**
   - 구체적 수치 (85% 감소)
   - 실현 가능한 기술
   - 명확한 작동 원리

### 심사위원 관점 고려

**그들이 보는 것:**
- 첫 5초: "이거 뭐지?" → 흥미 유발
- 10-20초: "어떻게 작동?" → 이해도
- 마지막 5초: "기억에 남는 메시지" → 임팩트

**그들이 평가하는 것:**
- 영상 기술력 (전문성)
- 아이디어 참신함 (창의성)
- 실행 가능성 (현실성)
- 사회적 가치 (의미)

### 최종 제출 시 고려사항

**파일 제출:**
- 파일명: 명확하고 전문적
- 포맷: 요구사항 정확히 준수
- 백업: 여러 버전 보관

**함께 제출 (가능하다면):**
- 1페이지 콘셉트 설명서
- 기술 구현 설명 (간단히)
- 사회적 임팩트 수치

## 💡 시간이 부족할 때 우선순위

1. **Apple "The Greatest"** (장애인 스포츠)

   - 메시지: "장애는 한계가 아니다"

   - 임팩트: 감성적 스토리텔링
   - 기술: 단순하지만 효과적
   - 메시지: "기술로 추억 보존"

3. **Dove "Real Beauty"**
   - 임팩트: 사회적 메시지
   - 기술: 다큐멘터리 스타일
   - 메시지: "진짜 아름다움"

- **Apple처럼**: 진정성 있는 노인 표현 (과장 없이)
